# Универсальная ролевая система

Посмотреть правила системы можно [здесь](система.md).

Идея системы состоит в том, чтобы предоставить мастеру и игрокам полную свободу для воображения.

## Требования к системе

- Воплощает любую пришедшую в голову историю
- Балансирует между простотой использования и высокой конфигурируемостью

## Как использовать

Идея состоит в том, что сначала пользователи системы придумывают всю нарративную составляющую, а затем гибко встраивает её в систему. Таким образом, система не определяет игру, а лишь помогает её провести, являясь инструментом рассказчиков и игроков.
